Dr. Santos' Student Registry

Description
This program is a simple student registry system written in C. It allows the user to store and manage up to 10 student records while the program is running.

Features
- Add a new student
- Display all students
- Search a student by ID
- Search students by GPA
- Search students by major
- Count students in a selected major

Concepts Used
- Arrays
- Functions
- Loops
- Conditional Statements
- scanf() and printf()
- String handling using fgets() and strcmp()

How to Compile

Using GCC:
gcc main.c -o registry

Run:
registry

or

./registry
